【*】程序简介 基于STM32F103ZET6的开发板 + HAL库 + 汇总板载资源

-工程名称：STM32F103ZET6_HAL_SDK 
-实验平台: 基于普中 STM32F103ZET6 玄武 开发板 
-HAL库版本：V1.1.4

-github厂库路径：https://github.com/hxq0818/F103ZET6_HAL_SDK.git或者git@github.com:hxq0818/F103ZET6_HAL_SDK.git

GPIOB_5			LED1

BEEP------>
GPIOB_8			BEEP
key-exti----->
GPIOA_0			KEY_UP
GPIOE_4			KEY0
